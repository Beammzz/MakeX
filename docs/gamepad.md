# `gamepad` --- รีโมทคอนโทรล (จอยสติ๊ก)

ฟังก์ชันหลักและเมธอดของโมดูล `gamepad`

## คำอธิบายรีโมทคอนโทรล

รีโมทคอนโทรลมีลักษณะดังภาพด้านล่าง:

(ไม่มีรูปภาพตามที่ระบุ)

## ฟังก์ชันที่เกี่ยวข้อง

### `get_joystick(joystick_pos)`

รับค่าคันโยกจอยสติ๊ก, ค่าที่คืนกลับมาอยู่ในช่วง `100 ~ -100` , ซ้ายเป็นบวกขวาเป็นลบ, บนเป็นบวกล่างเป็นลบ, พารามิเตอร์:

- `joystick_pos` เป็นสตริง (ชนิดข้อความ), โดยที่:
  - `"Lx"`: แกน x ฝั่งซ้าย
  - `"Ly"`: แกน y ฝั่งซ้าย
  - `"Rx"`: แกน x ฝั่งขวา
  - `"Ry"`: แกน y ฝั่งขวา

### `is_key_pressed(button)`

ตรวจสอบว่าปุ่มบนจอยถูกกดหรือไม่, คืนค่าเป็นบูลีน, โดย `True` หมายถึงปุ่มถูกกด, `False` หมายถึงปุ่มไม่ได้ถูกกด

- `button` เป็นสตริง (ชนิดข้อความ), โดยที่:
  - `"R1"`: R1 ฝั่งขวา
  - `"R2"`: R2 ฝั่งขวา
  - `"L1"`: L1 ฝั่งซ้าย
  - `"L2"`: L2 ฝั่งซ้าย
  - `"N1"`: เลข 1
  - `"N2"`: เลข 2
  - `"N3"`: เลข 3
  - `"N4"`: เลข 4
  - `"Up"`: D-pad ขึ้น
  - `"Down"`: D-pad ลง
  - `"Left"`: D-pad ซ้าย
  - `"Right"`: D-pad ขวา
  - `"+"`: ปุ่มบวก
  - `"≡"`: ปุ่มเมนู
  - `"L_Thumb"`: กดที่คันโยกซ้าย
  - `"R_Thumb"`: กดที่คันโยกขวา
