# `gyro_sensor` --- เซ็นเซอร์ไจโรสโคปบนบอร์ด

ฟังก์ชันหลักและเมธอดของโมดูล `gyro_sensor`

## คำอธิบายเซ็นเซอร์ไจโรสโคปบนบอร์ด

ระบบพิกัดของไจโรสโคปบนบอร์ดมีลักษณะดังภาพด้านล่าง:

(ไม่มีรูปภาพตามที่ระบุ)

## ฟังก์ชันที่เกี่ยวข้อง

### `get_pitch()`

ดึงค่ามุมก้มเงย (Pitch angle) (แกน X) ของท่าทาง, หน่วย: °, ช่วงข้อมูลที่ส่งคืนคือ `-180 ~ 180`

### `get_roll()`

ดึงค่ามุมม้วน (Roll angle) (แกน Y) ของท่าทาง, หน่วย: °, ช่วงข้อมูลที่ส่งคืนคือ `-180 ~ 180`

### `get_yaw()`

ดึงค่ามุมหัน (Yaw angle) (แกน Z) ของท่าทาง, หน่วย: °, ช่วงข้อมูลที่ส่งคืนคือ `-32768 ~ 32767`, เนื่องจากโมดูลไจโรสโคปบนบอร์ดเป็นเซ็นเซอร์ 6 แกนไม่มีเข็มทิศอิเล็กทรอนิกส์ ดังนั้นในความเป็นจริงมุมหันจึงเป็นเพียงการรวมค่าความเร็วเชิงมุมของแกน Z ซึ่งมีข้อผิดพลาดสะสม หากต้องการรับมุมหันที่แท้จริง API นี้ไม่เหมาะสมที่จะใช้

### `is_shaked()`

ตรวจสอบว่าโมดูลไจโรสโคปบนบอร์ดถูกเขย่าหรือไม่, คืนค่าเป็นบูลีน, โดยที่ `True` หมายถึงถูกเขย่า
